Banker Box Generator
A web-based die cut pattern generator for creating banker-style boxes with lids. Generate SVG patterns ready for laser cutting or manual crafting.
Features

Configurable Dimensions - Set interior width, depth, and height in millimeters
Separate Lid - Adjustable lid wall height with automatic clearance for fit
Double-Wall Construction - Inner panels fold back for reinforced walls
Optional Tabs & Slots - Glueless assembly with interlocking tabs, or simple folded construction
Structural Corner Wings - Adjustable wing lengths to prevent overlap on narrow boxes
Material Thickness Compensation - Fold lines offset to account for material thickness
Laser-Ready Export - SVG files with explicit mm units for accurate scaling
Two Export Modes:

Screen mode: Red fold lines for easy visualization
Laser mode: Dashed fold lines as actual cut segments



Usage

Open index.html in a web browser
Adjust dimensions and settings in the left panel
Preview the patterns in the right panel
Export separate SVG files for the bottom box and lid

Parameters
Main Dimensions

Inside Width/Depth/Height - Interior dimensions of the box
Lid Wall Height - Height of the lid walls
Wing Length - Length of structural corner wings (bottom box)
Lid Wing Length - Length of structural corner wings (lid)

Advanced Settings

Material Thickness - Thickness of your material (affects tab depth and fold offsets)
Tab Width/Spacing - Size and distribution of interlocking tabs
Assembly Clearance - Gap for easier assembly and fold compensation
Wing Corner Inset - Inset at wing corners for cleaner folds
Dash Length/Gap - Pattern for laser score lines
